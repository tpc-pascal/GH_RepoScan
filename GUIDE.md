1. Truy cập [GitHub Settings](https://github.com/settings/profile)
2. [Developer settings](https://github.com/settings/apps) > Personal access tokens > [Tokens (classic)](https://github.com/settings/tokens)
3. Generate new token > Generate new token (classic) > Name > Scopes: Repo > Generate token
4. Truy cập [Groq Console](https://console.groq.com/keys) > API Keys > Create API Key
5. Upload TXT của GitHub và Gemini
